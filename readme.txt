This is instruction from https://forum.farmanager.com/viewtopic.php?f=11&t=7592:
1. Download ConsoleControl plugin for x86/x64 https://plugring.farmanager.com/plugin.php?l=en&pid=886.
2. Put it on Far manager plugin directory.
3. Open far manager as adminitrator.
4. Run command (you version of ConCtrl may be different)
conctrl:/Load="C:\Program Files\Far Manager\Plugins\ConCtrl.far3.x64.1.4\BlueLikeScreenshots.pal"
5. Far должен изменить свою раскраску (пока она неправильная, см. далее). Затем откройте свойства окна Far-a ("клик" на системной иконке в заголовке окна -> "Свойства"), на закладке "Цвета" можно заметить, что там теперь нестандартный набор цветов. Ничего не изменяя нажмите "ОК". После этого обновятся свойства ярлыка или внесутся данные в реестр для окна Far-а.
